## Bienvenue sur mon projet Symfony

Ce site permet de vendre des livres (roman, bd, manga)

Il est possible d'ajouter des produits avec un multistepform, de modifier un produits, de regarder les details, et de le supprimer, avec les bonnes restrictions pour chaque rôle.
On peut d'ailleur exporter un produit en fichier CSV.

Il est aussi possible d'ajouter des utilisateurs et des clients, de les modifier et de les supprimer, tout en ayant une vérification sur les rôles.

## Commandes

Voici les commandes disponible : 

```php bin/console app:add-client``` Vous permet d'ajouter un nouveau client

```php bin/console app:import-products``` Vous permet d'importer un fichier csv (Il faut que le fichier soit dans le dossier Public et qu'il s'appelle "produits.csv", vous pouvez tout simplement modifier celui qui est existant)
